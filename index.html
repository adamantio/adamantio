Adamantio by adamantio

Opensource framework for cross-platform mobile development

Adamant is an opensource framework for web developers, allows them to build native mobile applications, cross-platform, using web technologies such as HTML, Javascript and CSS. This framework is based on Titanium Mobile, in contrast, Adamantio is completly opensource
